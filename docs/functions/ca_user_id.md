# CA_USER_ID

## Description

Returns the ChemApp user ID associated with the active license.

## Syntax

```excel
=CA_USER_ID()
```

## Arguments

This function requires no arguments.

## Returns

A short string containing the user ID.

Example:

```text
XXXX
```

## Underlying ChemApp routine

This function wraps the ChemApp routine:
  
See: [TQGTID](https://gtt-technologies.de/ca-doc/index.html#tqgtid)
