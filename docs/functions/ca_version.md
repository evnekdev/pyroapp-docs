# CA_VERSION

## Description

Returns the version number of the compiled ChemApp library; some functions might not be available if the version number is too old.


## Syntax

```excel
=CA_VERSION()
```

## Arguments

This functions requires no arguments.

## Returns

An integer number.

Example:

```text
710
```
